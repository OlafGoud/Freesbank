# Requirements

## MUST
- Kop kan bewegen over 3 assen (x, y, z)
- Kop kan materiaal toevoegen of verwijderen
- De positie van de kop kan bepaald worden door sensoren.
- Er kan een bestand om uit te voeren op de machine geladen worden
- Er word een variant van grbl gebruikt.



## SHOULD
- Kop kan bewegen over 4 assen (x, y, z, x rotation)

## COULD
- LCD scherm om te kijken hoever die is?

# Scope
De scope van de opdracht bevat het aansturen met code van de freesbank doormiddel van een vorm van grbl. Hier nog een sensor aan toevoegen om de exacte locatie van de kop te kunnen bepalen. (En simpele elektra voor de motoren aan te sturen.)
