# manual

## usage

### Supported commands
#### G commands

**Fast straight movement**
- G0, (x, y, z, f), no cutting
- G1, (x, y, z, f, e), cutting

**Arc movement**
- G2, (x, y, z, i, j, r), clockwise
- G3, (x, y, z, i, j, r), counter clockwise

#### M commands
- M1 -> stop instantly (not implmenented)
